# Lasagna-U-Net

This repository provides the code for ''Lasagna-U-Net: Revisiting U-Shape Architecture for Histopathology Image Segmentation''.

## Datasets

- GlaS: https://academictorrents.com/details/208814dd113c2b0a242e74e832ccac28fcff74e5
- KPIs: https://sites.google.com/view/kpis2024
- ISIC: https://challenge.isic-archive.com/data/
- BUSI: https://scholar.cu.edu.eg/?q=afahmy/pages/dataset
